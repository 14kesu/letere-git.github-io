Markdown语法详解

一小时快速上手，程序员必备的Markdown语法，轻松编写规范文档。

什么是Markdown？

Markdown 致力于使阅读和创作文档变得容易。

Markdown 是一种轻量级的标记语言，设计初衷是让人们可以使用简单的文本格式来书写和编辑文档，同时具有一定的格式化效果，通过使用Markdown语法，用户可以快速地将纯文本转化为格式化的文档，例如标题、列表，链接，图片等。 Markdown 语法简单直观，易于学习和使用，逐渐成为了各种平台上书写文档的常用格式。甚至还可以实现 Markdown 文档对 PDF、Word 等格式的快速转换。

Markdown 的文件通常以.md 或.markdown 为扩展名。

有序/无序列表和勾选框

无序列表

  • 我是第一项任务

  • 我是第三项任务

使用*（+、-）和一个空格符号表示列表。（不可以混用，会被分为不同的列表）

有序列表

引用文本

在语段前面加上一个> 表示这是我从别人的文章摘抄过来的。
如果要换行的话遵循正常文本的换行规则，但是不能直接空一行，否则会认为是另一段文本

或者在每一段前面加上尖括号，源代码比较清晰。

当然也可以嵌套其他的内容，代码块，列表，引用等。

超链接

初级用法

直接将网站贴上去之后显然不够美观

https://www.bilibili.com/

所以我们可以将超链接，直接将超链接的载体左右两边加上方框，在右方框后面加上小括号，小括号贴上链接

如：前往bilibili官网观看视频。

高级用法

当一段话有数量较多的超链接时，我们可以在每个文本的右括号在加一组方括号，括号内给其起名，然后再文章末写上方框与起的名字，冒号，冒号后面空一格写上链接。

如：前往bilibili或者腾讯视频观看视频

而且变量是可以复用的。

角注（索引，注释）

想要购买请前往官网1

再要注释的地方后边加上方框，方框内写上^与变量名，变量名的赋值与超链接相同。

图片的插入

图片的插入只能以链接（网络或本地）的形式上传。一般推荐以网络形式上传。

首先打上一个感叹号，方括号，小括号

方括号内写名字，小括号写链接。这个图片不能手动调整大小。部分软件可以再链接后面写上"",中间加上文字表示鼠标悬浮时的备注。

本地上传可以直接写图片名字。也可以使用与超链接方式相同的变量方式命名。

数学公式的插入与排版

美元符号的使用

使用两个美元符号，中间加上内容，不可换行。

$x = 1 + y$

或者在文本的上下两行分别打两个美元符号，中间全部内容变为数学公式的样式。

$$ \frac{1}{x}*5 $$

分数

在右斜杠后面加上frac{}{}，两个花括号表示分子和分母。
